## Clean Room Implementation
LSW is developed using **clean-room reverse engineering principles**:
1. Study Microsoft's architecture and patterns (this fork)
2. Document the concepts and API behaviors
3. Implement LSW independently from scratch
4. Test compatibility with real Windows applications
